### Naming convention for resource files

| Filetype          | Fileextension |
|:-----------------:|:-------------:|
| Basic Texture     | *.png, *.jpg  |
| Texture Atlas     | *.atlas       |
| Animation         | *.atlas.anim  |
| Animation Details | *.anim        |
| Vertex Shader     | *.vert        |
| Fragment Shader   | *.frag        |
