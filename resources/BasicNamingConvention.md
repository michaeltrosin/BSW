### Naming convention for resource files

|     Filetype      | Fileextension |
| :---------------: | :-----------: |
|   Basic Texture   |    \*.png     |
|       _-"-_       |    \*.jpg     |
|   Texture Atlas   |   \*.atlas    |
|     AnimationHandler     |    \*.anim    |
|   Vertex Shader   |    \*.vert    |
|  Fragment Shader  |    \*.frag    |
